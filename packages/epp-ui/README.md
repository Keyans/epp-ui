# epp-ui

## 注册一个tnpm账号

```
tnpm adduser
```

### 登陆注册好的tnpm账号

```
tnpm login
```

### 打包对应的lib文件夹

```
npm run lib
```

### 发布到对应的tnpm上

```
tnpm publish
```
